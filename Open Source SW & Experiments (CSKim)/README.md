# 오픈소스SW및실습
### Open Source SW & Experiments

#### 2020-1 / 김차성 교수님   
- 오픈소스의 정의와 오픈소스 프로젝트에 대해 배웁니다.   
- Pygame Zero(pgzero)를 이용한 게임 제작, 라즈베리파이를 이용한 LAMP 연동 및 개발 실습이 진행됩니다.
- 2학년 과목이지만 고학년이 많이 수강하며, Ubuntu 서버, SQL Database 사용, 웹서버 이해를 하고 있어야 강의 듣기 수월합니다.
- 시험 문제는 강의 초반에 설명하는 강의 PPT에서 대부분 출제됩니다. (우분투의 역사, MQTT의 사용법, 오픈소스의 정의 등)
- 간단한 손코딩 문제와 코드의 결과를 작성하는 문항도 존재합니다.
- 레포트 비중이 높습니다.
