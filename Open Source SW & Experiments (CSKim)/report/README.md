# Report
- 레포트는 총 6번으로, pgzero를 이용한 게임 만들기 3번, 라즈베리파이를 이용한 과제 3번이 출제됩니다.
- 주제는 매년 바뀌지 않으며, 구현 목표가 조금씩 수정됩니다.
- 레포트의 설명과 소스 코드는 각 하위 폴더에 정리되어 있습니다.
- 코드의 구현을 위해서는 LAMP, Python 등의 이해가 필요합니다.

### 순서
1. [ConnectFourGame](https://github.com/accko199806/hufsces/tree/master/Open%20Source%20SW%20%26%20Experiments%20(CSKim)/report/ConnectFourGame) : Javascript로 구현된 Connect4 게임을 pgzero로 구현하기
2. [PongGame](https://github.com/accko199806/hufsces/tree/master/Open%20Source%20SW%20%26%20Experiments%20(CSKim)/report/PongGame) : Javascript로 구현된 Pong 게임의 AI를 개선하여 pgzero로 구현하기 
3. [Pacman2](https://github.com/accko199806/hufsces/tree/master/Open%20Source%20SW%20%26%20Experiments%20(CSKim)/report/Pacman2) : Pacman2 게임의 Ghost AI 개선하기 
4. [RaspStatusPublish](https://github.com/accko199806/hufsces/tree/master/Open%20Source%20SW%20%26%20Experiments%20(CSKim)/report/RaspStatusPublish) : MQTT로 라즈베리파이 상태 Publish하기
5. [RaspStatusSaveMysql](https://github.com/accko199806/hufsces/tree/master/Open%20Source%20SW%20%26%20Experiments%20(CSKim)/report/RaspStatusSaveMysql) : Publish된 라즈베리파이 상태를 Subscribe하여 MySQL DB에 저장하기
6. [RaspStatusShowWeb](https://github.com/accko199806/hufsces/tree/master/Open%20Source%20SW%20%26%20Experiments%20(CSKim)/report/RaspStatusShowWeb) : 실시간으로 저장되는 MySQL DB 데이터를 불러와 웹상에서 그래프로 보여주기
